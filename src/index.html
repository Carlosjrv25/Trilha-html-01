O que são as Tags de Texto?
As tags de Texto são instruções que informam o navegador como formatar o texto no HTML(corpo do site).

Pincipais exemplos de tags de Texto:

  • < h1 > - Define o titulo de maior

  • < h2 > - Define o titulo

  • < h3 > - Define o titulo

  • < h4 > - Define o titulo

  • < h5 > - Define o titulo
  • < h6 > - Define o titulo de menor

As tags < h1 ao h6 > definem os titulos no site por tamanho assim como pdoemos ver no exemplo citado acima.

O que são as tags de listas ordenadas?

As Tags de listas ordenadas são representadas pela tag "ol" que siginifica "ordered list" onde o "li" é usado para representar um item que faz parte de uma lista.

  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3
  4. Exemplo 4
OBS: No exemplo acima , podemos perceber que a lista tem uma ordem numérica no qual se da o nome de listas ordenadas.
Exemplo básico de como se escreve o código da tag de lista ordenada:

< ol >

< li >primeiro item< /li >
< li >segundo item< /li >
< li >terceiro item< /li >

< /ol >

O que são as tags de listas nâo ordenadas?

As Tags de listas ordenadas são uma lista de itens sem ordem rígida que não recebem ordem numéricas .

  • Exemplo 1
  • Exemplo 2

  • Exemplo 4
OBS: No exemplo acima , podemos perceber que a lista não tem uma ordem numérica onde a lista esta serparada por um marcador e por isso o nome de listas não ordenadas.
As Tags de listas não ordenadas são representadas pela tag "ul" que siginifica "unordered list" onde o "li" é usado para representar um item que faz parte de uma lista.

Exemplo básico de como se escreve o código da tag de lista ordenada:

< ul >

< li >primeiro item< /li >
< li >segundo item< /li >
< li >terceiro item< /li >

< /ul >
