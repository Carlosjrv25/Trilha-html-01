Tags de Texto

O que são as Tags de Texto?
As tags de Texto são instruções que informam o navegador como formatar o texto no HTML(corpo do site).

Pincipais exemplos de tags de Texto:

  • < h1 > - Define o titulo de maior

  • < h2 > - Define o titulo

  • < h3 > - Define o titulo

  • < h4 > - Define o titulo

  • < h5 > - Define o titulo
  • < h6 > - Define o titulo de menor

As tags < h1 ao h6 > definem os titulos no site por tamanho assim como pdoemos ver no exemplo citado acima.
